insert_image_to_md

方法一： 组织化管理资源文件

  1. 将配置文件_config.yml中的参数post_asset_floder默认值改为 true。
  2. 在hexo目录下执行npm install hexo-asset-image –save , 即安装本地上传图片的插件。
  3. 再执行hexo n filename, 此时在_posts文件夹中生成一个filename.md和一个filename文件夹。
  4. 把图片复制到filename文件中，此时通过常规的markdown语法它不会出现在首页上，在Hexo3中，使用下面标签插件代码即可：

方法二：最简单的html图片标签

  1. 在source文件中建立images文件。
  2. 在md文件中写入html图片标签：<img src=”images/图片名.jpg”>。
